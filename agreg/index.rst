Un métadéveloppement d'informatique
-----------------------------------

Pour la leçon sur la NP-complétude, j'avais comme développement :

    *Étant donné une liste de développements possibles par leçon, déterminer un nombre minimal de développements pour couvrir toutes les leçons est un problème NP-difficile.*

Voici d'ailleurs `un TP sur le sujet </_static/set-cover.pdf>`_. On y apprend que **42** développements suffisent à garantir 2 développements par leçon pour l'option D (informatique).
